# Aplicativo de educação financeira com assistente virtual 
Aplicativo mobile para controle de finanças com uma assistente virtual Financeira, Mrs. Finanças. 

Tecnologias Utilizadas:

- Expo CLI
- Firestore
- Dialogflow
- Firebase Authentication
